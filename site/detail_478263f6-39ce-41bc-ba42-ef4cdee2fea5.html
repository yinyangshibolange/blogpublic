轻松实现远程“局域网”

分类：折腾的网络
作者：hongyu
2024-06-11 16:17:00

zerotier

zerotier是一个多系统的软件终端，可以用来搭建远程“局域网”，只要两个设备同时安装zerotier并加入了同一个“网络”，那么这两个设备就可以互相访问。

为什么选择zerotier而不是内网穿透（蒲公英等）或frp

  1. 内网穿透的话免费的速度太慢，而速度快的又太贵
  2. frp需要有公网ip的服务器（其实zerotier也需要公网ip的服务器，不过zerotier默认提供，且具有较大带宽）

安装

windows

只要下载msi安装安装即可

linux

运行一下代码即可

curl -s https://install.zerotier.com | sudo bash

群晖

使用矿神第三方套件安装，注意这个套件需要修复才能使用，使用教程在套件详情的链接中

创建“局域网”

首先需要注册登录zerotier官网，然后点击Create A Network 按钮，即可创建“局域网”

加入“局域网”

windows

运行zerotier, 右键zerotier托盘，点击join network，输入network id ，点击join即可。

linux

zerotier-cli join yournetworkId

群晖

同linux

激活ip

加入完的设备还没有分配ip，需要进入zerotier官网的网络面板下的“members”面板，可以看到加入的设备，选中左边的多选框即可，name和Description选填（用来区分设备），然后Managed IPs中的ip就是设备对应的ip，之后访问这个ip即可。
